## Download ##

`git clone https://github.com/UstymUkhman/YetAnotherZombieHorror.git`

`cd YetAnotherZombieHorror`

`yarn`

## Develop ##

`yarn start:web`

`yarn start:app`

`yarn lint:js`

`yarn lint:css`

## Test ##

`yarn test:run`

`yarn test:cover`

`yarn serve`

## Build ##

`yarn build:web`

`yarn build:app`

## Distribute ##

`yarn pack:app`

  - **Linux:** `yarn dist:app -l`

  - **Windows:** `yarn dist:app -w`
